# Grille de fiabilite des sources

## Tier 1 — Sources primaires et institutionnelles

| Type | Exemples | Fiabilite |
|------|----------|-----------|
| Publications scientifiques peer-reviewed | Nature, Science, Lancet, PubMed | Tres haute |
| Organismes publics de recherche | CNRS, INSERM, BRGM, NASA, USGS | Tres haute |
| Agences gouvernementales | INSEE, ANSES, EFSA, FDA, EMA | Tres haute |
| Organisations internationales | OMS, ONU, GIEC, AIE/IEA, OCDE | Tres haute |
| Rapports officiels et donnees ouvertes | data.gouv.fr, Eurostat, World Bank | Tres haute |

## Tier 2 — Medias de reference et fact-checkers

| Type | Exemples | Fiabilite |
|------|----------|-----------|
| Fact-checkers certifies IFCN | AFP Factuel, Les Decodeurs (Le Monde), Snopes, Full Fact | Haute |
| Medias de reference FR | Le Monde, AFP, Reuters, France Info | Haute |
| Medias de reference INT | BBC, AP News, NYT, The Guardian | Haute |
| Presse scientifique specialisee | Sciences et Avenir, Pour la Science, Nature News, New Scientist | Haute |
| Presse economique/technique | Les Echos, L'Usine Nouvelle, Financial Times | Haute |

## Tier 3 — Sources secondaires fiables

| Type | Exemples | Fiabilite |
|------|----------|-----------|
| Encyclopedies collaboratives | Wikipedia (verifier les sources citees) | Moyenne-haute |
| Medias generalistes | France 24, 20 Minutes, BFMTV | Moyenne |
| Presse regionale | Republicain Lorrain, Ouest-France | Moyenne |
| Blogs d'experts identifies | Blogs de chercheurs, The Conversation | Moyenne |
| Communiques de presse | A verifier contre les sources primaires | Moyenne |

## Tier 4 — Sources a traiter avec prudence

| Type | Exemples | Fiabilite |
|------|----------|-----------|
| Reseaux sociaux | LinkedIn, Twitter/X, Facebook | Faible — verifier la source originale |
| Medias d'opinion | Contrepoints, Mediapart (editos), Brut | Faible — biais editorial |
| Sites thematiques militants | Sites pro/anti-nucleaire, pro/anti-OGM | Faible — biais ideologique |
| Communiques d'entreprises | PR de startups, annonces corporate | Faible — biais commercial |
| Agregateurs et reprises | MSN, Yahoo News, Orange Actu | Variable — verifier la source originale |

## Tier 5 — Sources non fiables

| Type | Exemples | Fiabilite |
|------|----------|-----------|
| Sites complotistes | Identifiables par le ton, absence de sources | Nulle |
| Blogs anonymes sans sources | Pas d'auteur identifie, pas de references | Nulle |
| Contenus sponsorises deguises | Publi-reportages, native advertising | Nulle — conflit d'interet |
| IA generative non sourcee | ChatGPT, articles generes sans verification | Nulle — hallucinations possibles |

## Signaux d'alerte sur une source

- Absence d'auteur identifie
- Pas de date de publication
- Aucune source ou reference citee
- Ton alarmiste ou superlatif ("revolutionnaire", "unique au monde", "jackpot")
- URL suspecte ou nom de domaine trompeur
- Article identique copie-colle sur plusieurs sites
- Financement ou conflit d'interet non declare
