# Patterns de biais et manipulation a detecter

## Biais de presentation

| Pattern | Description | Exemple |
|---------|-------------|---------|
| Clickbait / sensationnalisme | Titre exagere par rapport au contenu | "Jackpot! La France decouvre un tresor de $92 milliards" |
| Superlatifs non justifies | "le plus grand", "unique au monde", "historique" sans comparaison | "La plus grosse reserve au monde" sans mentionner que c'est une estimation |
| Chiffres impressionnants sans contexte | Gros nombre sans echelle de comparaison | "46 millions de tonnes" sans dire que c'est theorique |
| Faux exclusif | "revelation", "on vous cache" alors que l'info est publique | "Ce que les medias ne disent pas..." |

## Biais de raisonnement

| Pattern | Description | Comment detecter |
|---------|-------------|-----------------|
| Correlation = causalite | Deux evenements correles presentes comme lien de cause a effet | Verifier si une etude interventionnelle existe |
| Cherry-picking | Selection de donnees qui supportent la these, omission du reste | Comparer avec les donnees completes |
| Generalisation abusive | Un cas particulier extrapole a l'ensemble | Verifier la taille d'echantillon et la representativite |
| Appel a l'autorite | "Les scientifiques disent..." sans nommer ni citer | Verifier qui a dit quoi exactement |
| Homme de paille | Deformer la position adverse pour mieux la refuter | Comparer avec la position reelle |
| Pente glissante | "Si X alors inevitablement Y puis Z catastrophique" | Verifier chaque etape du raisonnement |

## Biais d'omission

| Pattern | Description | Comment detecter |
|---------|-------------|-----------------|
| Omission de contexte | Fait vrai mais prive de son contexte | Rechercher le contexte complet |
| Omission d'incertitude | Estimation presentee comme certitude | Chercher les mots "estime", "pourrait", "si confirme" dans la source primaire |
| Omission de risques | Benefices sans mentionner les inconvenients | Rechercher "[sujet] risques limites critiques" |
| Omission temporelle | Pas de date, ou date masquee pour une vieille info | Verifier la date de publication et des faits |

## Biais de source

| Pattern | Description | Comment detecter |
|---------|-------------|-----------------|
| Conflit d'interet | L'auteur ou le financeur a un interet dans la conclusion | Chercher qui finance, qui vend |
| Echo chamber | Plusieurs articles qui se citent mutuellement sans source primaire | Remonter la chaine de citations |
| Communique de presse deguise | Article journalistique qui est en fait un CP d'entreprise | Comparer avec le CP original |
| Etude financee par l'industrie | Recherche payee par une entreprise qui beneficie des resultats | Verifier la section "funding" des papers |

## Checklist de detection rapide

Avant de conclure, verifier :
- [ ] Le titre correspond-il au contenu de l'article ?
- [ ] Les chiffres cites ont-ils une source identifiable ?
- [ ] Les citations sont-elles en contexte ?
- [ ] Les experts cites sont-ils du bon domaine ?
- [ ] Les limites et incertitudes sont-elles mentionnees ?
- [ ] Y a-t-il un conflit d'interet potentiel ?
- [ ] L'info est-elle reprise par des sources Tier 1-2 ?
