# Sources fiables — Priorite de consultation

## Tier 1 — Gold standard (privilegier)

| Source | Type | Acces |
|--------|------|-------|
| Cochrane Library | Revues systematiques | cochranelibrary.com |
| PubMed / PMC | Meta-analyses, RCT | pubmed.ncbi.nlm.nih.gov |
| Examine.com | Resumes bases sur preuves avec grades | examine.com |

## Tier 2 — Sources institutionnelles

| Source | Type | Acces |
|--------|------|-------|
| EFSA | Avis scientifiques EU | efsa.europa.eu |
| FDA | Positions reglementaires US | fda.gov |
| ANSES | Avis securite sanitaire FR | anses.fr |
| OMS / WHO | Recommandations mondiales | who.int |
| EMA | Agence europeenne du medicament | ema.europa.eu |

| Source | Type | Acces |

| McGill OSS | Analyse critique | mcgill.ca/oss |
| NHS | Fiches sante UK | nhs.uk |
| Science-Based Medicine | Blog critique | sciencebasedmedicine.org |

## Tier 4 — Verification et signalement

| Source | Usage |
|--------|-------|
| Retraction Watch | Verifier si une etude a ete retractee |
| Semantic Scholar | Nombre de citations, impact |
| Google Scholar | Citations croisees |

## Sources a eviter ou a signaler

- Blogs sans revision par les pairs
- Sites commerciaux vendant le produit etudie
- Etudes financees exclusivement par l'industrie concernee (signaler le conflit)
- Preprints non repliques (bioRxiv, medRxiv) — utilisables comme signal mais pas comme preuve
