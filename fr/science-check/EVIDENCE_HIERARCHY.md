# Grille des niveaux de preuve

## Hierarchie (du plus fort au plus faible)

| Niveau | Type d'etude | Poids | Utilisation dans le verdict |
|--------|-------------|-------|----------------------------|
| 1a | Meta-analyse de RCT / Revue Cochrane | Tres fort | Peut justifier "CONFIRME" ou "REFUTE" seul |
| 1b | RCT large (n>200) double aveugle | Fort | Confirme si replique par un 2e RCT |
| 2 | RCT petit (n<200) ou simple aveugle | Modere | "PARTIELLEMENT CONFIRME" si coherent |
| 3 | Etude de cohorte / observationnelle | Faible | Correlation, pas causalite — signaler |
| 4 | Etude sur animaux | Tres faible | "PREMATURE" — non transposable directement |
| 5 | In vitro (cellules en labo) | Minimal | "PREMATURE" — mecanisme plausible seulement |
| 6 | Avis d'expert / tradition / temoignages | Nul | "NON PROUVE" — ne constitue pas une preuve |

## Regles d'attribution du verdict global

| Situation | Verdict |
|-----------|---------|
| Meta-analyse positive + pas de contradiction majeure | CONFIRME |
| Plusieurs RCT positifs mais pas de meta-analyse | PARTIELLEMENT CONFIRME |
| Resultats contradictoires entre etudes de qualite comparable | PARTIELLEMENT CONFIRME (signaler la controverse) |
| Uniquement etudes animales / in vitro | PREMATURE |
| Aucune etude de qualite trouvee | NON PROUVE |
| Meta-analyse negative ou majorite de RCT negatifs | REFUTE |

## Biais a signaler systematiquement

- **Financement industrie** : etudes financees par le fabricant du produit etudie
- **Petit echantillon** : n < 50 dans un RCT
- **Duree courte** : etude < 4 semaines pour un effet a long terme
- **Population non representative** : uniquement hommes jeunes, uniquement athletes, etc.
- **Endpoints de substitution** : marqueur biologique au lieu de resultat clinique reel
- **Publication bias** : que des resultats positifs publies dans le domaine
