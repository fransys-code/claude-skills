# Source Reliability Grid

## Tier 1 — Primary Sources and Institutional Sources

| Type | Examples | Reliability |
|------|----------|-------------|
| Peer-reviewed scientific publications | Nature, Science, Lancet, PubMed | Very high |
| Public research organizations | CNRS, INSERM, BRGM, NASA, USGS | Very high |
| Government agencies | INSEE, ANSES, EFSA, FDA, EMA | Very high |
| International organizations | WHO, UN, IPCC, IEA, OECD | Very high |
| Official reports and open data | data.gouv.fr, Eurostat, World Bank | Very high |

## Tier 2 — Reference Media and Fact-checkers

| Type | Examples | Reliability |
|------|----------|-------------|
| IFCN certified fact-checkers | AFP Factuel, Les Decodeurs (Le Monde), Snopes, Full Fact | High |
| Reference French media | Le Monde, AFP, Reuters, France Info | High |
| International reference media | BBC, AP News, NYT, The Guardian | High |
| Specialized scientific press | Sciences et Avenir, Pour la Science, Nature News, New Scientist | High |
| Business/tech press | Les Echos, L'Usine Nouvelle, Financial Times | High |

## Tier 3 — Reliable Secondary Sources

| Type | Examples | Reliability |
|------|----------|-------------|
| Collaborative encyclopedias | Wikipedia (verify cited sources) | Medium-high |
| General media | France 24, 20 Minutes, BFMTV | Medium |
| Regional press | Republicain Lorrain, Ouest-France | Medium |
| Blogs by identified experts | Researcher blogs, The Conversation | Medium |
| Press releases | To be verified against primary sources | Medium |

## Tier 4 — Sources to Handle with Caution

| Type | Examples | Reliability |
|------|----------|-------------|
| Social media | LinkedIn, Twitter/X, Facebook | Low — verify original source |
| Opinion media | Contrepoints, Mediapart (editorials), Brut | Low — editorial bias |
| Activist thematic sites | Pro/anti-nuclear sites, pro/anti-GMO sites | Low — ideological bias |
| Corporate communications | Startup PR, corporate announcements | Low — commercial bias |
| Aggregators and reprints | MSN, Yahoo News, Orange Actu | Variable — verify original source |

## Tier 5 — Unreliable Sources

| Type | Examples | Reliability |
|------|----------|-------------|
| Conspiracy sites | Identifiable by tone, lack of sources | None |
| Anonymous blogs without sources | No identified author, no references | None |
| Disguised sponsored content | Advertorials, native advertising | None — conflict of interest |
| Unverified AI-generated content | ChatGPT, articles generated without verification | None — hallucinations possible |

## Red Flags for a Source

- No identified author
- No publication date
- No sources or references cited
- Alarmist or superlative tone ("revolutionary", "unique in the world", "jackpot")
- Suspicious URL or misleading domain name
- Identical article copied and pasted across multiple sites
- Undisclosed funding or conflict of interest
