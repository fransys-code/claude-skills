# Verdict Scale

## Main Verdicts

| Verdict | Meaning | When to Use |
|---------|---------|-------------|
| TRUE | The claim is factual and correct | Confirmed by 3+ independent reliable sources |
| MOSTLY TRUE | The substance is correct but with important nuances | Real fact but missing context or oversimplification |
| EXAGGERATED | Based on real facts but amplified or distorted | Inflated figures, unjustified superlatives, clickbait on true substance |
| MISLEADING | Mix of true and false, or true but presented to mislead | Cherry-picking, out of context, correlation presented as causation |
| PREMATURE | Too early to conclude, research ongoing | Preliminary studies, unconfirmed estimates, immature technology |
| UNVERIFIED | Impossible to confirm or deny with available sources | No primary source found, claim is not verifiable |
| MOSTLY FALSE | The essence of the claim is incorrect | Majority of facts contradicted by reliable sources |
| FALSE | The claim is factually incorrect | Contradicted by solid evidence, debunked by fact-checkers |

## Confidence Level of Verdict

| Level | Meaning |
|-------|---------|
| HIGH | 5+ reliable convergent sources, primary source verified |
| MEDIUM | 3-4 sources, or sources of variable quality |
| LOW | Few available sources, topic poorly covered |

## Attribution Rules

- An article can be TRUE in substance but EXAGGERATED in presentation — both are flagged
- If sub-claims have different verdicts, the overall verdict reflects the weighting
- A single wrong figure in an otherwise true article = MOSTLY TRUE (not FALSE)
- Clickbait headlines on correct content = flag the headline/content gap
- If the primary source contradicts the media reprint = MISLEADING
